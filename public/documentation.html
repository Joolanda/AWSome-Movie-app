CloudFlix API Documentation

This is a backend "movie" application build using Node.js. This application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create list of their favorite movies. User authentication and authorization is implementated using JWT tokens.

Sl No Request Method type Endpoint Url Query Parameters Request data format Response data format
1 Return a list of all movies GET /movies None None Returns a JSON array of all movies
2 Return data about a single movie by TITLE GET /movies/:Title :Title None Returns a JSON object of movie matching the title passed in the url
3 Return data about a single movie By GENRE GET /movies/genre/:Genre :Genre None Returns a JSON object holding movie data which matches genre name.
4 Return a list of movies BY DIRECTOR NAME /directors/:Name GET :Director None Returns a JSON object of all the movies matching by the director name that was passed in the url
5 Allow new user to register POST /users None A JSON Object holding data about the new user.

Example:
{
"Username": "user1",
"Password": "password1",
"Email": "user1@gmail.com"
"Birthday": "09/02/2000"
}
Returns a JSON Object holding data about the new user created, with an id.

Example:
{
"_id": XXXXX,
"Username": "user1",
"Password": "password1",
"Email": "user1@gmail.com"
"Birthday": "09/02/2000"
}
6 Allow users to update their user info PUT /users/:Username :Username A JSON Object holding data the user information along with field to update. Returns a JSON object with updated user data
7 Allow users to add a movie to their list of favorites /users/:UserID/movies/:MovieID POST :UserID, :MovieID None Returns a JSON object with updated user data
8 Allow users to remove a movie from their list of favorites /users/:UserID/movies/:MovieID DELETE :UserID, :MovieID None Returns a JSON object with updated user data
9 Allow existing user to deregister /users/:Username DELETE :Username None Returns a text confirming that the user was deleted successfully.
